6. Architecture & Sécurité

Sécurité

Chiffrement des données

AES-256-CBC pour données patients sensibles. Mots de passe Argon2id. HTTPS forcé. Headers sécurité (X-Frame, X-XSS, CSP).

RGPD

Conformité

Consentement explicite, page mentions légales complète, audit log, durées de conservation, droits d'accès/rectification/suppression.

Base de données

MySQL / 8 tables

users, settings, contacts, appointments, available_slots, blocked_dates, prevention_articles, audit_log.

Hosting

Hostinger ready

PHP natif + MySQL. .htaccess configuré. Pas de framework lourd. Compatible hébergement mutualisé.
